# 2차원 배열 입력

## 코드

```python
import sys

_input = sys.stdin.readline

row, col = map(int, _input().rstrip().split())

graph = []
for _ in range(row):
  graph.append(list(map(int, _input().rstrip())))

print(graph)
print(graph[0][0])
```

## 실행

```shell
python main.py
4 6
101111
101010
101011
111011
```
